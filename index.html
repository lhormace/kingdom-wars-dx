Kingdom Wars DX

ケルトの意匠を思わせる深緑と金の戦場で、王国軍を率いて敵城へ進軍するレトロ風ライン戦術アクションです。王・兵士・騎士・魔法使い・僧侶の隊列を守りながら、捕虜救出と聖剣回収を経てラスボスを倒してください。

☘ 永き石輪と森の加護をまとい、古き王国の道を進め ☘

戦いの誓い

  • 敵を倒して敵城へ進み、ラスボス撃破後に城へ入るとステージクリア。
  • 捕虜を助けると味方が増え、3人ごとに騎士へ昇格します。
  • 聖剣エクスカリバーを拾うと強敵に大ダメージを与えられます。

戦場の作法

移動
矢印キー / WASD
魔法
F キー / 発射軌跡を表示
再開始
R キー または右上ボタン
開始
Enter / Space / 開始ボタン（進軍中は一時停止 / 再開）

戦列と聖遺物 / 凡例

画面内の見た目に寄せたアイコンで、初期隊列は 王 → 兵/騎士 → 大魔法使い → 僧侶 の順です。

  • 王：前線で戦う主人公
  • 兵士：数で支える基本戦力
  • 騎士：高火力の精鋭
  • 大魔法使い：後方から光線で支援。MPは戦場表示でも確認できます
  • 僧侶：祈りで自動回復を担当。王の自然回復も支援します
  • 龍ラスボス：4x4 の巨体で城門前を守る最終敵
  • エクスカリバー：取得時に強い発光エフェクトが発生
